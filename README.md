# Qu::Thermo

A thermodynamics library for calculating DNA/DNA binding energy

## Installation

Add this line to your application's Gemfile:

    gem 'qu-thermo'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install qu-thermo

## Usage

`tm = Qu::Thermo::Hybrid.new(seq_1, seq_2).tm`
